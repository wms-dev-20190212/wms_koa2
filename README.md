# vue-koa2————基于Vue2实现的单页面应用

## 概述



项目地址 `https://github.com/gyx6644932/koa2`


## 实现功能

* 对话功能——在项目中接入了图灵聊天机器人，可以与列表中的每个人物进行对话。

* 左滑删除——左滑删除相关消息。

* 搜索页面——点击右上角搜索按钮，能够进入搜索页面，输入对应的单词或者数字，动态查找好友。


***

## 技术栈

*  koa2
*  vue-cli
*  vue2
*  vue-router
*  vuex
*  axios
*  stylus
*  webpack2
*  muse-ui
